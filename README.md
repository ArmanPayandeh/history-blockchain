# History of Blockchain

A comprehensive documentation website chronicling the evolution of blockchain technology from foundational papers to modern developments.

## Features

- **Chronological Archive**: Browse blockchain papers and research organized by year
- **Papers Blog**: Stay updated with recent publications and developments
- **Multi-language Support**: Available in English and Persian (فارسی)
- **Modern Documentation**: Built with Docusaurus for fast, responsive, and searchable content
- **Open Access**: All shared papers are freely available or properly licensed

## Prerequisites

- Node.js >= 20.0
- npm (comes with Node.js)

## Installation

```bash
npm install
```

## Local Development

```bash
npm start
```

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

## Build

```bash
npm run build
```
